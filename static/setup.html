Connect Claude to your tools

Add MarketingMCP as a custom connector in Claude so it can manage your ads, analytics, and SEO.

1

Open Claude Settings

Go to claude.ai and click your profile icon in the bottom-left corner, then select Settings.

🤖 Claude → Profile icon → ⚙ Settings
2

Go to Connectors

In the Settings panel, find and click Connectors in the left sidebar. This is where you manage external MCP integrations.

⚙ Settings → 🔌 Connectors
3

Add a Custom Connector

Click the "Add custom connector" button. You'll see a form asking for the MCP server URL.

➕ Look for the "Add custom connector" button — it's usually at the top of the Connectors page.
4

Paste the MCP URL

Copy the URL below and paste it into the connector URL field. This is your personal MCP server endpoint.

Create & Connect
